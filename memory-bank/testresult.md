# testresult.md — 테스트 결과 누적 기록

테스트 결과는 최신순(위)으로 누적 기록한다.

## 테스트 항목 정의

| ID | 테스트 항목 | 합격 기준 |
|----|------------|-----------|
| T-01 | 네이버 서버 시간 동기화 | 서버 Date 헤더 파싱 성공, offset_ms 계산값 출력 |
| T-02 | 오프셋 수동 조정 | 슬라이더 이동 시 일치율(%) 실시간 반영 |
| T-03 | Z 키 좌표 등록 | Z 키 입력 시 마우스 위치 (x, y) 정상 저장 |
| T-04 | 좌표 재등록 | 기존 슬롯 값이 새 좌표로 교체됨 |
| T-05 | 카운트다운 표시 | 목표 시각까지 `HH:MM:SS.mmm` 형식으로 감소 |
| T-06 | 매크로 시작 조건 | 좌표 3개 미등록 시 버튼 비활성화 |
| T-07 | 정밀 클릭 타이밍 | 목표 시각 대비 오차 ±10ms 이내 |
| T-08 | 좌표 3개 순서 클릭 | 좌표1 → 좌표2 → 좌표3 순으로 클릭 확인 |
| T-09 | 서버 요청 실패 폴백 | 네트워크 차단 시 경고 표시 후 로컬 시간 사용 |
| T-10 | 과거 시각 입력 차단 | 현재보다 이전 시각 입력 시 시작 불가 안내 |
